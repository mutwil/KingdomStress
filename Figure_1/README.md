# Figure 1 -- Kingdom-wide stress transcriptome atlas

Scripts that generate the data-driven panels of Figure 1 (dataset overview and pipeline validation across 36 plant species and 9 stress types).

The assembled figure is available as [`Figure 1.pdf`](Figure%201.pdf).

## Scripts

| Script | Panels | Description |
|--------|--------|-------------|
| `fig1a_phylo_heatmap.py` | A | Manual cladogram of 36 species coupled with a heatmap of control/treatment pair counts per species x stress, plus a Total column |
| `fig1c_marker_heatmap.py` | D | Stress-marker validation: GO-verified canonical Arabidopsis markers, mapped via OrthoFinder to orthologs in all species, log2FC pulled from the DEG table and visualized as a stripplot with stress-colored boxplots |

Panels B (pipeline schematic) and C (in-house phenotype photos) are not generated by scripts.

## Run order

The two scripts are independent and can be run in any order. Neither builds a shared cache; both read the master DEG table directly.

## Input data

Paths are hard-coded near the top of each script. Update them to point to your local copies:

```python
DEG_FILE          = "/path/to/kingdom_stress_dict v3.csv"
ORTHOGROUPS_FILE  = "/path/to/Orthogroups.txt"
GO_FILE           = "/path/to/ATH_GO_GOSLIM.txt"
```

| File | Used by |
|------|---------|
| `kingdom_stress_dict v3.csv` | both |
| `Orthogroups.txt` | `fig1c_marker_heatmap.py` |
| `ATH_GO_GOSLIM.txt` | `fig1c_marker_heatmap.py` |

## Methods summary

- **Control/treatment pair counting (Panel A)**: unique `species + bioproject + experiment` combinations. Some BioProjects are shared across species, so the species prefix is required for correct counts.
- **Phylogeny (Panel A)**: manual cladogram encoded as nested Python tuples following APG IV topology for angiosperms and established Viridiplantae relationships. Rendered with matplotlib (no external tree library).
- **Marker gene curation (Panel D)**: ~40 canonical Arabidopsis stress markers were compiled from the literature, then intersected with TAIR `ATH_GO_GOSLIM.txt` entries having experimental evidence codes (IMP, IDA, IEP, IGI) for a GO term matching the assigned stress category. 35 markers passed.
- **Ortholog mapping**: each verified Arabidopsis marker is mapped to its OrthoFinder orthogroup, and all member genes (across all 36 species) are used as orthologs. Gene IDs are matched at locus level after stripping transcript suffixes (`.1`, `.2`).
- **log2FC summarization (Panel D)**: one data point per species x marker x experiment, taking the ortholog with the largest |log2FC| when an orthogroup contains multiple paralogs per species.
